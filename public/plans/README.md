# Reference plan images

Section 5 of the app renders these five files. Until they are present, the app
shows a styled "Plan image pending" placeholder that names the expected path;
drop the matching PNG here and redeploy and it renders automatically.

| File | Card |
| ---- | ---- |
| `floor-plan-f1.png` | First floor plan (compact) |
| `floor-plan-f1-full.png` | Proposed First Floor Plan |
| `unit-c1-2br.png` | Unit Plan — C1 · 2BR/2BA · 1,125 SF |
| `unit-b1-3br.png` | Unit Plan — B1 · 3BR/3BA · 1,350 SF |
| `site-construction-plan.png` | Site Construction Plan |

Source originals were uploaded in the authoring chat session under
`/mnt/user-data/uploads/` (a cloud-container path not available on the local
machine that built this repo), so they were not copied in at build time.
